# E-commerce-travel-website

A responsive Travel Website developed using HTML, CSS, and JavaScript.  
This project provides an attractive platform where users can explore travel destinations and view travel packages through a clean and interactive interface.

## 📌 Project Overview
The Travel Website is designed to showcase different tourist destinations with a visually appealing layout. The website focuses on responsive design, easy navigation, and a user-friendly experience. It demonstrates fundamental front-end development skills using HTML, CSS, and JavaScript.

## 🚀 Features
- Responsive design for different screen sizes
- Attractive travel destination sections
- Smooth navigation between pages
- Interactive elements using JavaScript
- Modern UI design using HTML and CSS

## 🛠️ Technologies Used
- HTML– Structure of the website  
- CSS– Styling and layout design  
- JavaScript – Interactive functionality
